# Plateforme de Défis Sportifs

## Description

Ce projet consiste en une plateforme en ligne de défis sportifs permettant aux utilisateurs de s’inscrire et de participer à différents challenges physiques (ex : course, pompes, squats, etc.).

L’objectif principal est de motiver les utilisateurs à pratiquer une activité sportive régulière à travers des défis simples et accessibles.

## État d’avancement du projet
- Système d’authentification (inscription / connexion / déconnexion)
- Gestion des défis (création, affichage, participation)
- Tableau de classement des participants
- Interface responsive avec Bootstrap
- Utilisation de PHP (PDO)
- Base de données fonctionnelle (hébergée sur un service externe si nécessaire)
